# Hospital Management System

A Hospital Management System built using React, Node.js, Express, and MongoDB.

## Features

- Doctor Management
- Patient Management
- Appointment Scheduling
- Dashboard
- Reports

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- Vite

### Frontend

npm run dev
